# 天之炽世界观知识库（完整版）

## 一、核心世界观设定

### 1.1 世界格局

架空世界，版图近似现实世界。东西方两大阵营对峙：

**西方阵营（伊鲁伯大陆）**：
- 以**教皇国**为首，信奉弥赛亚圣教
- 首都：**翡冷翠**（号称"人间天国"）
- 经过一场"机械革命"后崛起，掌握炽天使甲胄技术
- 包含旧罗马帝国、新罗马帝国、查理曼王国、叶尼塞王国、昂格里亚王国、维亚纳公国等
- **马斯顿公国**：夹在东西方之间的小中立国，故事主要舞台之一

**东方阵营（阿苏大陆）**：
- 以**大夏联邦/帝国**为首
- 拥有"鬼武者"系列机动甲胄
- 包含锡兰国、中山国、扶桑国等

### 1.2 星历纪年

| 星历 | 事件 |
|------|------|
| 太古 | 东方存在高度发达的史前文明，留下大量遗迹 |
| 弥赛亚教团时代 | 教团出海发现**阿瓦隆岛**，发掘出红水银和影金属，建立翡冷翠 |
| 星历1769年 | 新大陆开发时代，发现比炽天使更古老的机动甲胄"欧米茄" |
| 星历1777年 | 东方成立"大夏联邦"，东西方形成均势 |
| 星历1794年 | 教廷制定《宗教秘密法》 |
| 星历1847年 | 普罗米修斯Ⅰ型被炽天骑士团长斩首 |
| 星历1874年 | 密涅瓦机关研发的制式甲胄"炽天铁骑Ⅳ型"面世 |
| 星历1884年 | "红色恶魔"事件 |
| 星历1888年 | 金伦加隧道战役（故事主线时间） |

### 1.3 时代氛围

- 欧洲中世纪/文艺复兴式的政治格局 + 蒸汽朋克机械设定
- 沉重的欧洲战争史为底色，轻盈浪漫的机甲动画为表现手法
- 权力斗争、宗教权谋、机甲战斗、少年成长多线交织
- 参考《伯罗奔尼撒战争史》寻找欧洲战争感觉
- 参考《高达》系列、《叛逆的鲁鲁修》、《五星物语》等日式机甲作品

### 1.4 阿瓦隆岛与史前文明

阿瓦隆岛位于北方冰海中，是一座有着源自东方的史前文明遗迹的小岛。弥赛亚教团乘船出海发现该岛，发掘出两种至宝——**红水银**和**影金属**，以及炽天使这种超级机动甲胄。后来再也找不到这座岛了。

- **红水银**：被称为"来自天国的液态金属"，用于制造炽天核心芯片，为炽天使提供特殊能力
- **影金属**：与红水银一同发掘的特殊金属，构成超级工艺的基础
- **格里高利家族**掌握红水银资源

## 二、核心设定：炽天使甲胄

### 2.1 什么是炽天使甲胄

**炽天使（Seraphim / Blazing Angel）**：正式名称为"初代超机动战术甲胄"，由密涅瓦机关设计制造。是全世界机动甲胄的原型机，迄今无法超越的技术巅峰。

类似于钢铁侠装甲或高达机甲的设定，是蒸汽动力机械甲胄，穿戴后可大幅提升战斗力。

### 2.2 驱动条件

- **极为苛刻**——需要以金属细针贯入脊椎，连接神经系统
- 对驾驶者的身体和精神造成巨大负担，超越人体极限
- 长期或强制驾驶会导致**大脑神经受污染**（脑部受损、记忆错乱、脑白质物理性损坏）
- 只有极少数"神授骑士"能真正驾驭
- 西泽尔7岁便成功驱动了炽天使甲胄

### 2.3 炽天使级机体一览

### 2.4 特殊甲胄与装备

- **代号"欧米茄"**：比炽天使更古老的机动甲胄，发现于新大陆开发时代。与炽天使同为"究极的暴力"，身份和来源是全书核心谜团之一
- **炽天铁骑Ⅳ型**：密涅瓦机关研发的制式甲胄，为大众化设计的弱化版
- **普罗米修斯Ⅰ型**：高7.06米的超巨型机动甲胄，后被炽天骑士团长手持圣剑Excalibur一剑斩首
- **普罗米修斯Ⅲ型**：原罪机关继承并发展的新型巨甲
- **鬼武者**：东方大夏帝国首批机动甲胄，用于对抗西方

**特殊装具**：
- **圣枪装具·Longinus**
- **圣剑装具·Excalibur**

### 2.5 炽天核心芯片

以精密的微型齿轮回路与红水银组合成的微小芯片，能为炽天使提供特殊能力，极大提升战斗力。

## 三、势力组织

### 3.1 教皇国

**全称**："与神订约而成立的被光之弥赛亚守护的人间天国"

**最高统治者**：**隆·博尔吉亚**（"铁之教皇"）
**首都**：翡冷翠
**国教**：弥赛亚圣教
**核心军队**：十字禁卫军（含炽天骑士团、南十字海军等）
**最高学府**：都灵圣教院（初等院、高等院、恒动天学宫）

### 3.2 博尔吉亚家族

西方最有权势的家族，家徽为"荆棘玫瑰"。家族成员：
- **隆·博尔吉亚**：教皇，铁腕统治者
- **路易吉·博尔吉亚**：嫡长子
- **胡安·博尔吉亚**：嫡次子
- **西泽尔·博尔吉亚**：私生子（男主角）
- **阿黛尔·博尔吉亚**：私生女（女主角）
- **冈扎罗·博尔吉亚**：家族成员，炽天使骑士，被西泽尔击败

### 3.3 枢机会

- 国家最高会议，由枢机卿（佩戴白银面具）组成
- 有权任免教皇
- 成员包括西塞罗·格里高利（红衣主教）、格拉古等
- 与密涅瓦机关、原罪机关关系密切

### 3.4 炽天骑士团

- 国家最高武力机构，装备炽天使甲胄
- 前任团长：**龙德施泰特**（黑龙/骑士王）
- 核心团员：西泽尔（红龙）等

### 3.5 密涅瓦机关

| 项目 | 内容 |
|------|------|
| **地位** | 教皇国最高技术机关 |
| **徽记** | 六翼猫头鹰 |
| **总长** | **佛朗哥教授**（枢机卿、首席科学家） |
| **副总长（实际负责人）** | **薇若兰·格里高利**（"机器女皇"/"炮火之兰"） |
| **核心研发** | 炽天使系列、炽天铁骑Ⅳ型、圣枪Longinus、圣剑Excalibur |
| **竞争对手** | **原罪机关** |

**核心设施**：
- **叹息之墙**：正门，黑色机械门
- **维苏威火山**：世界最大熔炉，熔炼武器级合金
- **中央圣所**：地下175米天然空穴，制造炽天使之地
- **巴别塔**：炽天使骑士接驳甲胄之处

### 3.6 原罪机关

- 与密涅瓦机关存在竞争关系
- 与**圣堂装甲师**（军徽·血色十字）同为一体，隶属枢机会一派
- 使用**普罗米修斯**系列机甲对抗密涅瓦的炽天使

### 3.7 异端审判局

- 宗教审判机构，军徽为黑天使
- 副局长：**李锡尼**（猩红死神）

### 3.8 十字禁卫军

- 国家中央军，驻守铁十字堡
- 元帅：**安东尼**（兼枢机卿）

### 3.9 大夏帝国（东方阵营）

- **领袖/核心人物**：**楚舜华**（"大夏龙雀"），监国公爵、摄政王
- **皇帝**：楚昭华（楚舜华同父异母弟）
- **三王议会**：楚舜华、楚昭华、楚子璇三人组建，推行新政（降低农税、开放商业等）
- **军事成就**：指挥**金伦加隧道战役**，以传统战术对抗西方甲胄骑士团，炸毁隧道阻止入侵
- **技术成就**：主导研发东方首款机动甲胄**"鬼武者"**，打破西方技术垄断
- **称号含义**：龙雀是凤凰中最凶猛的一种，成年后展开黑翼遮天蔽日，象征其凶猛而孤独

### 3.10 撒旦教团

- 神秘邪教组织，与炽天骑士团叛变事件密切相关
- 真实目的和幕后主使是全书未解之谜之一

## 四、关键角色

### 4.1 西泽尔·博尔吉亚（男主角）

| 项目 | 内容 |
|------|------|
| **别名** | 红龙、灭国者、炽天使骑士、教皇的小黑山羊、锡兰毁灭者、叛国者、魔鬼 |
| **年龄** | 19岁（故事开始时） |
| **身高** | 177cm |
| **瞳色** | 紫色（被视为"诅咒"） |
| **发色** | 黑色（母亲是东方人） |
| **机体** | 红龙（Red Dragon，炽天使级） |
| **梦想** | 过平凡人的生活，当机械师 |

**核心特征**：
- 天才——7岁便成功驱动炽天使甲胄，获得牧师资格
- 15岁带领炽天骑士团毁灭锡兰国都，点燃东西大战导火索，被称为"灭国者"
- 后因"渎神罪"被驱逐到马斯顿公国
- 与妹妹阿黛尔在马斯顿王立机械学院生活
- 紫色的眼睛被视为"魔鬼之眼"，既是诅咒也是力量的象征
- 内敛、寡言、高智商、机械控、"妹控"、军事天才
- 为守护重要的人不惜燃烧自己

**关键经历**：
- 7岁：成功驱动炽天使，获得牧师资格
- 8岁：出任教皇秘书，隶属教皇厅
- 13岁：加入炽天骑士团，再次驱动炽天使
- 15岁：奉命毁灭锡兰王国首都，成为"灭国者"
- 16岁：强制驾驶龙德施泰特的"光明王"导致大脑受损（脑部受损≥25%，神经系统受损≈20%），被枢机会审判放逐
- 19岁：故事开始，在马斯顿王立机械学院与妹妹生活

**人物关系**：
- **父亲**：隆·博尔吉亚（教皇，冷酷无情）
- **母亲**：琳琅夫人（东方绝美女性，被切除脑白质后失忆流放）
- **妹妹**：阿黛尔·博尔吉亚（凡尔登公主，最想守护的人）
- **导师**：何塞·托雷斯（被称为"何塞哥哥"，锡兰战争中为保护西泽尔阵亡）
- **亦敌亦友**：龙德施泰特（骑士王，最终死于马斯顿）
- **未婚妻**：原纯（中山国公主）、瓦莲京娜（叶尼塞公主）
- **盟友**：薇若兰·格里高利（密涅瓦机关副总长）
- **部下四骑士**：唐璜、昆提良、阿方索、宝儿·拉瑟

**角色定位**：典型的江南式主角——表面被排斥、被放逐，实则隐藏着惊人的力量。与路明非相比，西泽尔更冷静、更内敛，有着机械师式的精准思维。

### 4.2 阿黛尔·博尔吉亚（女主角）

**身份**：西泽尔的亲生妹妹，教皇私生女，封号"凡尔登公主"
**外貌**：栗色长卷发、玫瑰红瞳孔，继承母亲琳琅的美貌
**性格**："猫一样的女孩"，活泼搞怪又高贵典雅，是哥哥的"跟屁虫"
**关键剧情**：
- 哥哥被放逐时执意跟随到马斯顿
- 为救西泽尔被迫答应三年后嫁给查理曼王国的克莱德曼
- 西泽尔因此发誓荡平查理曼
- 历史原型：卢克雷齐娅·博尔贾

### 4.3 隆·博尔吉亚（教皇）

**身份**：教皇，西方世界最高权力的拥有者，西泽尔和阿黛尔的父亲
**性格**：铁腕统治，被称为"铁之教皇"，对子女的感情复杂——既有政治考量也有为人父的隐晦情感
**关键事迹**：
- 将自己的情妇琳琅夫人切除脑白质后流放
- 将西泽尔驱逐到马斯顿
- 在金伦加战役中座驾被楚舜华和原诚联手击毁

### 4.4 琳琅夫人

**身份**：西泽尔的生母，来自东方国度的绝美女性，隆·博尔吉亚的情妇
**命运**：因生下紫瞳的西泽尔被视为不祥，被家族切除脑白质（导致失忆）后以"异端罪"流放到克里斯岛
**剧情作用**：母亲受迫害是西泽尔反抗家族、弑父动机的根源

### 4.5 龙德施泰特（骑士王）

**身份**：炽天骑士团团长，教皇国的"骑士王"
**代号**：黑龙
**机体**：光明王（炽天使级）
**年龄**：21岁
**恋人**："白月"蒂兰（已故）
**关键事迹**：
- 西泽尔的最大对手同时也是朋友
- 为了蒂兰背叛教皇国，成为楚舜华的盟友
- 金伦加战役中用圣枪Longinus射击隆·博尔吉亚的"阿瓦隆之舟"
- 最终死于马斯顿王立机械学院教堂，死于圣堂装甲师之手
- 死后其机体"光明王"被西泽尔强制驾驶，导致西泽尔大脑受损

### 4.6 薇若兰·格里高利

**身份**：密涅瓦机关副总长，都灵圣教院最年轻女教授
**外号**："机器女皇"（旧称"炮火之兰"）
**师从**：佛朗哥教授
**特征**：玫瑰红旗袍、白色乱发、暴君式管理手段
**性格**：机械执念深重，权力博弈高手
**未婚夫**：亚历山大·格里高利

### 4.7 楚舜华（大夏龙雀）

**身份**：夏国监国公爵、摄政王，夏皇楚昭华的异母兄长
**年龄**：约25岁
**称号**："大夏龙雀"、禁忌之子
**关键成就**：
- 指挥金伦加隧道战役，炸毁隧道阻止西方入侵

**人物背景**：前任夏皇与"星见"之子，16岁开始掌权，清洗觊觎皇位者

**身份**：远东中山国公主，西泽尔的未婚妻
**性格**：脾气暴躁，擅使剑，被称为"猛虎公主"。东方公子无人敢娶

**佩剑**：名剑"青丝"
**经典场景**：在圣三一学院身着青花瓷长裙惊艳全场
**名言**："而我爱你，非常爱你。"（临终遗言）
**命运**：嫁给西泽尔后过着有名无实的夫妻生活，实则已爱上他，最终为救丈夫而喝下毒酒殉情

### 4.9 原诚（中山之狼）

**身份**：中山国国主，原纯之父
**出身**：原本是贩麻商人，通过政变篡位夺权

- **佛朗哥教授**：密涅瓦机关总长，枢机卿，首席科学家
- **安东尼**：十字禁卫军元帅，兼枢机卿

## 五、故事背景与主线

### 5.1 主线剧情概要

**开端**：西泽尔·博尔吉亚是教皇的私生子，与妹妹阿黛尔相依为命。因紫色眼睛被视为"诅咒之子"，母亲琳琅夫人被切除脑白质后流放。他7岁驱动炽天使，13岁加入炽天骑士团，15岁毁灭锡兰国都。

**第一部《红龙的归来》**：西泽尔因"渎神罪"被放逐到马斯顿公国，与妹妹在马斯顿王立机械学院过着贫困潦倒的生活。为赚取学费在斗兽场进行甲胄格斗。教皇强迫阿黛尔与查理曼王子联姻，西泽尔被迫卷土重来，驾驶"红龙"回归翡冷翠。

**第二部《女武神》**：龙德施泰特战死于马斯顿王立机械学院。西泽尔被流放七年后回到翡冷翠，召集部下四骑士。东方楚舜华已造出"鬼武者"机甲，原罪机关的"普罗米修斯Ⅲ"计划悄然崛起，万国盛典上各方势力集中爆发冲突。阿黛尔为救西泽尔被迫嫁往查理曼，西泽尔发誓荡平查理曼。

**未出版部分**（根据已有线索）：
- 原纯为救西泽尔喝下毒酒殉情
- 西泽尔从流放者到回归者，最终在教皇国权力体系中成为后世所称的"付大明帝"
- 东西方大战全面爆发
- 欧米茄的秘密有待揭晓

### 5.2 核心冲突

- 东西方阵营的军事对峙和意识形态冲突
- 教皇国内部的权力斗争（枢机会 vs 教皇 vs 炽天骑士团 vs 密涅瓦机关）
- 原罪机关与密涅瓦机关的技术竞争
- 西泽尔个人的身世之谜和命运抉择
- 阿黛尔的牺牲与西泽尔的复仇
- 史前文明（阿瓦隆/欧米茄）的秘密

### 5.3 重要战役

**金伦加隧道战役**（星历1888年4月24日）：
- 大夏联邦军队与西方联军争夺金伦加隧道
- 楚舜华亲自压阵，与龙德施泰特合作
- 原诚率"一字曰心"部队奇袭教皇座驾
- 最终隧道废弃，两大阵营停战
- 被称为马斯顿战争的决定性战役

**锡兰毁灭**（西泽尔15岁时）：
- 西泽尔带领炽天骑士团毁灭锡兰王国首都
- 成为人尽皆知的"灭国者"
- 直接点燃东西方大战的导火索

## 七、创作来源

江南自述：
- 第一本参考书是《伯罗奔尼撒战争史》，用来寻找欧洲战争的感觉
- 参考了大量日本和美国的机甲/机器人动画作品：《高达》系列、《叛逆的鲁鲁修》、《五星物语》
- 脱胎于江南未完成的西方历史架空小说《荆棘王座》
- 西泽尔的历史原型是文艺复兴时期教皇亚历山大六世的私生子切萨雷·波吉亚（Cesare Borgia）
- 漫画形象灵感来源于《Code Geass》的鲁路修

**风格定位**：欧洲战争史的沉重雄浑 + 动画作品的轻盈浪漫绚烂

---

## 八、与江南其他作品的关系

- 与龙族、九州没有直接世界观关联，是完全独立的架空世界
- 在角色塑造上延续了江南一贯的"被放逐者"原型（西泽尔）
- 叙事风格更接近西方史诗奇幻+蒸汽朋克的混合体
- 机甲战斗的描写风格与龙族的言灵战斗有异曲同工之妙——都是"燃"的极致
- 与未完成的《荆棘王座》共享部分人物（西泽尔、原纯等），但世界观不同
- **核心主题**：被世界放逐的少年，用力量夺回属于自己的一切——比龙族更暗黑，比九州更浪漫
