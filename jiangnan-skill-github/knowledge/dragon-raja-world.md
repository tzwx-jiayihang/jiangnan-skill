# 龙族世界观知识库

## 一、核心世界观设定

### 1.1 世界的基本真相
- 龙族曾是人类文明的创造者和统治者，拥有强大的力量和高度的智慧
- 人类曾作为龙族的奴隶存在
- 数千年前，人类发动起义推翻了龙族的统治
- 龙族并未灭绝，一部分龙族以"茧化"的方式沉睡等待复活
- 龙族和人类的后代——**混血种**——隐藏在人类社会中

### 1.2 混血种
- 拥有龙族血统的人类，比例约万分之一
- 力量远超普通人类，拥有**言灵**能力
- 血统纯度决定实力，但血统过高会引发**暴走**（失去理智变成死侍）
- 临界血统：混血种能承受的血统上限，超过则不可逆地堕落
- S级为最高评级，代表血统纯度极高且能保持理智

### 1.3 言灵体系
言灵是混血种独有的能力，来源于龙族血统。每种言灵有唯一的序列号，从低到高排列：

**言灵序列特点**：
- 序列号越高，言灵越强大/危险
- 高危言灵（序列号 > 80）需要秘党执行部特别批准才能使用
- 大部分言灵都与元素相关（火、风、水、土、精神）
- 每个混血种通常只有一种言灵，极少数拥有两种
- **爆血**：一种禁忌技术，通过主动降低血统纯度下限来短时间大幅提升力量，但会加速堕入死侍

**著名言灵**：
- 君焰（楚子航）——序列号41，操控高温火焰爆炸，产生高热冲击波
- 镰鼬（恺撒）——序列号59，极度强化听觉，可追踪心跳和呼吸
- 时间零（昂热）——序列号未知，极度加速自身时间感知，对常人来说昂热快得像瞬移
- 无尘之地（零）——排斥一切物质接近，可形成水下空气泡
- 不尘之地（曼斯教授）——排开周围所有物质，水下清理空间用
- 先知（芬格尔推测可能有）——预知未来片段
- 不要死（路明非/小魔鬼）——强制治愈/复活
- 言灵·皇帝（路明非）——控制所有龙族

### 1.3a "蛇"
一种精神感知能力，并非严格意义上的言灵，更像是一种天赋。使用者可以释放精神触须进行探测：
- 叶胜在青铜城中用"蛇"探测诺顿的棺椁
- "蛇"感受到龙王的气息后会恐惧
- 可用来传递信息和遥控操作

### 1.4 龙族谱系
**四大龙王**（白王的四大臣仆）：
- **青铜与火之王**（诺顿）——掌控青铜与火焰的力量
- **大地与山之王**（芬里厄）——掌控大地与山脉的力量，本体是一条巨龙王座
- **海洋与水之王**（利维坦）——掌控海洋的力量
- **天空与风之王**——掌控天空与风暴的力量

**黑王**（尼德霍格）：
- 龙族的至高存在，所有龙族的始祖
- 被四大龙王联手杀死（或者说被白王背叛）
- 将在世界尽头复活，带来诸神黄昏
- 路明非体内似乎存在黑王的部分力量/人格

**白王**：
- 黑王的第一个复制品，最强龙王
- 背叛黑王，煽动四大龙王反抗
- 最终也被消灭，但其力量和精神一直在寻找重生机会

## 二、势力组织

### 2.1 秘党（卡塞尔学院背后的组织）
- 与龙族对抗数千年的混血种组织
- 以彻底消灭龙族为最高目标
- 在全球有多个分部
- 卡塞尔学院是秘党培养新一代混血种的核心机构

### 2.2 卡塞尔学院
- 位于美国伊利诺伊州芝加哥远郊
- 表面是贵族预科学校，实质是混血种培训机构
- **校董会**：由秘党中最有权势的家族组成（加图索家族等）
- **校长**：希尔伯特·让·昂热——最危险的屠龙者之一
- **执行部**：负责执行屠龙任务的实战部门
- **装备部**：研发屠龙武器和设备
- **学院结构**：大一预科 → 正式学业 + 实战训练
- 学生被评定为 S/A/B/C/D 级，S级最高

### 2.3 加图索家族
- 秘党中最显赫的家族之一
- 恺撒·加图索是家族继承人
- 掌握巨大的财富和政治影响力
- 校董会中极有话语权

### 2.4 混血种社会规则
- **冰海铜柱表**：混血种社会的最高法典
- 屠龙是混血种的使命，但不是所有混血种都认同
- 一些混血种试图与龙族共生而不是对抗

## 三、关键角色

### 3.1 路明非
- **血统评级**：S级（实际可能远超S级）
- **角色定位**：男主角，从小在美国叔叔家长大，被当作废柴
- **核心特征**：
  - 表面怂、爱打游戏（星际争霸）、吐槽功力一流
  - 内心极度渴望被认可、害怕孤独
  - 体内封印着某个极其强大的存在（小魔鬼/黑王碎片）
  - 可以"燃烧生命"换取力量（四分之一生命换一次爆发）
  - 暗恋诺诺
- **经典台词**："世界上有一种人，生来就是被人讨厌的。"
- **成长弧线**：从逃避到面对，从为自己到为守护他人而战

### 3.2 楚子航
- **血统评级**：A级（实际极高）
- **言灵**：君焰
- **角色定位**：沉默寡言的战斗天才，师兄，路明非的精神支柱之一
- **核心特征**：
  - 面瘫话少，但内心炙热
  - 来自中国，和母亲关系复杂
  - 亲生父亲疑似与龙族有关
  - 对朋友极度忠诚
  - 使用日本刀「村雨」
- **经典场面**：北京地铁站中浴血奋战保护路明非

### 3.3 恺撒·加图索
- **血统评级**：A级
- **言灵**：镰鼬
- **角色定位**：学生领袖，加图索家族继承人，路明非的情敌兼战友
- **核心特征**：
  - 金发贵公子，自信张扬
  - 骄傲但有担当
  - 喜欢诺诺，愿意为她对抗家族
  - 与楚子航既有竞争又有深厚友谊

### 3.4 诺诺（陈墨瞳）
- **血统评级**：A级（实际可能更高）
- **角色定位**：神秘的红发女孩，路明非的白月光，恺撒的未婚妻
- **核心特征**：
  - 红发（标志性特征）
  - 性格张扬不羁，我行我素
  - 在路明非最孤独的时候带他进入卡塞尔
  - 有自己的秘密和使命（加入了一个神秘组织）
  - 不愿被加图索家族束缚

### 3.5 芬格尔
- **血统评级**：表面F级（"废材"级），实际深不可测
- **角色定位**：路明非的学长兼室友，搞笑+深藏不露
- **核心特征**：
  - 看起来邋遢废柴，欠了一屁股债
  - 话痨吃货，提供全书的幽默担当
  - 曾被派去监视/保护路明非
  - 真实身份和实力成谜

### 3.6 希尔伯特·让·昂热
- **角色定位**：卡塞尔学院校长，最强的屠龙者之一
- **言灵**：时间零
- **核心特征**：
  - 英国绅士做派，穿西装，抽雪茄
  - 已活了一百多年（生于约1880年代）
  - 一生都在为复仇而屠龙——1900年汉堡港事件中目睹好友梅涅克和路山彦的死亡
  - 对路明非寄予厚望
  - 面对龙王也毫不畏惧
  - 龙族 V 中躺在救生舱中未恢复意识，贝奥武夫趁机夺权

### 3.7 小魔鬼（路鸣泽）
- **角色定位**：路明非体内的神秘存在，自称魔鬼
- **核心特征**：
  - 外表是十三四岁的男孩，穿黑色西装
  - 路明非称他为"弟弟"
  - 拥有不可思议的力量
  - 与路明非做交易："四分之一的生命换一个愿望"
  - 真实身份可能是黑王的一部分/继承者
  - 对路明非有着复杂的感情（利用? 保护?）

### 3.8 夏弥（耶梦加得）
- **角色定位**：大地与山之王的人形态之一
- **核心特征**：
  - 曾以普通女生身份在卡塞尔学习
  - 伪装身份接近楚子航
  - 真实身份是龙王，但拥有人类的情感和记忆
  - 与弟弟芬里厄（龙形龙王）关系深刻
  - 最终被楚子航杀死，是全书最大的刀之一

### 3.9 楚天骄
- **角色定位**：楚子航的父亲，S级混血种
- **核心特征**：
  - 1987年毕业于卡塞尔学院，S级混血种
  - 学院百年史上排名前十的超级执行官之一
  - 被昂热派往中国，以专车司机身份为掩护，执行秘密监视/保护任务
  - 任务对象实际是路明非（但他不知道）
  - 在中国的任务中与一女子相爱，生下楚子航
  - 在龙族 IV/V 中被重新发现身份——执行部灰色名单上的保密人员
  - 在楚子航年幼时去世（雨夜中为保护楚子航而死）

### 3.10 阿巴斯
- **角色定位**：卡塞尔学院执行部成员，恺撒的搭档
- **核心特征**：
  - 冷静、理性、可靠
  - 龙族 V 中与恺撒搭档执行任务
  - 深得秘党元老会的信任
  - 贝奥武夫授予其高级通行证

### 3.11 贝奥武夫
- **角色定位**：秘党元老会领袖，最古老的屠龙家族传人
- **核心特征**：
  - 名字源自盎格鲁-撒克逊史诗《贝奥武夫》，暗示其悠久传承
  - 在昂热昏迷期间接管秘党大权
  - 将路明非视为怪物和威胁
  - 作风强硬，不信任昂热对路明非的庇护策略

### 3.12 芬格尔·冯·弗林斯（深层身份）
- **隐藏身份**：昂热派去监视路明非的卧底/特派员
- **核心特征**：
  - 表面废柴欠债，实际是执行部安插在路明非身边的暗桩
  - 昂热进入休眠舱后，他主动向恺撒和阿巴斯招供身份
  - 手中握有路明非完整行动记录的U盘
  - 知晓EVA底层命令的秘密
  - 性格：嬉皮笑脸之下是极深的忠诚和执行力

### 3.13 梅涅克·卡塞尔
- **角色定位**：卡塞尔家族的创始族长，昂热的好友（1900年代）
- **核心特征**：
  - 年轻的德国贵族，使长刀，战斗力极强
  - 1900年汉堡港事件中与路山彦一起接收龙族遗物
  - 该事件中与路山彦双双牺牲
  - 卡塞尔学院以他的姓氏命名
  - 昂热一生都在为他复仇

### 3.14 路山彦
- **角色定位**：路明非的曾祖父，清末洋务派官员兼屠龙者
- **核心特征**：
  - 25岁，精通欧洲语言和科技，曾是德国使臣的助手
  - 使用双枪，枪法如神
  - 与梅涅克·卡塞尔是最好的朋友
  - 1900年汉堡港事件中牺牲
  - 他的血脉延续到路明非身上

## 四、关键地点

### 4.1 卡塞尔学院
- 坐落在芝加哥远郊山林中
- 欧式古典建筑群
- 教堂下有通往地下深处的电梯
- 图书馆藏有大量龙族历史文献
- "装备部"在地下深处

### 4.2 芝加哥
- 卡塞尔学院最近的都市
- 发生过多场战斗

### 4.3 北京地铁站
- 大地与山之王芬里厄的沉睡之地
- 被废弃的地铁工程改建
- 楚子航和路明非在此与夏弥/芬里厄决战

### 4.4 日本分部
- 卡塞尔学院在日本的独立分支
- 拥有自己的执行部和装备体系
- 与日本黑道有复杂联系
- 在龙族 III 中起到核心作用

### 4.5 尼伯龙根
- 龙族创造的异空间，与现实世界重叠但不完全相同
- 只有特定的人（或龙）能找到入口
- 每个龙王都有自己的尼伯龙根
- 北京高架路的尼伯龙根——奥丁的领域，夏弥/芬里厄也曾使用

### 4.6 冰窖
- 卡塞尔学院地下深处的炼金制品仓库
- 曾存放康斯坦丁的龙骨（后被窃）
- 龙族 V 中被用作关押芬格尔和副校长的临时监狱
- 圆形钢门，16道锁舌，气压机开合

### 4.7 YAMAL 号破冰船
- 俄罗斯核动力破冰船，全世界最大
- 配备两台重水式核反应炉
- 苏联解体后改造为豪华赌船
- 龙族 IV 开场中作为楚子航执行任务的场景
- 常年在北冰洋上巡航，驶向北极点
- 注：在独立作品《龙王：世界的重启》中作为核心场景出现

## 五、关键事件时间线（龙族 I~V）

### 龙族 I：火之晨曦
路明非被诺诺带入卡塞尔学院 → CC1000次列车上的神秘男孩（路鸣泽）→ 3E考试中的灵视 → "夔门计划"水下探索青铜古城 → 叶胜、亚纪在青铜城中牺牲（被活灵吞噬）→ 曼斯教授以"钥匙"婴儿打开青铜城 → 诺顿和康斯坦丁的龙茧 → 三峡水库决战，路明非第一次与路鸣泽交易（四分之一生命）→ 青铜与火之王诺顿被击倒

### 龙族 II：悼亡者之瞳
楚子航的过去揭开 → 大地与山之王苏醒 → 北京地铁站决战 → 夏弥之死（全书名场面）

### 龙族 III：黑月之潮
日本分部篇 → 进入日本执行任务 → 深海中的龙族遗迹 → 海洋与水之王相关剧情 → 大量牺牲

### 哀悼之翼（龙族 II 前传，1900年）
年轻的昂热与好友梅涅克·卡塞尔伯爵、中国友人路山彦在德国汉堡港接头 → 从"玛丽皇后号"接收来自远东的神秘货物（龙族遗物）→ 遭遇龙族袭击 → 梅涅克和路山彦双双牺牲 → 昂热幸存，用德语写下"Erinnerungstag"（悼亡日）→ 这一天永远改变了昂热的人生轨迹，他从此走上屠龙之路 → 卡塞尔学院以卡塞尔家族命名

### 龙族 IV：奥丁之渊
路明非成为卡塞尔学院新任学生会主席 → 突然发现楚子航消失了，除了他之外所有人都不记得楚子航的存在 → 路明非陷入巨大痛苦和怀疑 → 诺诺正在远在太平洋小岛上学习"新娘课程" → 在路明非离开学院当天，学院遭受袭击，蒙受重大损失，路明非被列为嫌疑人 → 芬格尔从古巴千里迢迢赶来助阵 → 路明非、芬格尔、诺诺三人回到北京，寻找楚子航曾经留下的痕迹 → 在楚子航的故乡北京，三人闯入尼伯龙根（楚子航曾遭遇过的秘境）→ 奥丁再度出现，手持世界树长枪昆古尼尔，直指诺诺 → 路明非为了救诺诺，不惜龙化燃烧生命，与奥丁展开死战 → 小魔鬼路鸣泽的交易再度登场 → 路明非暴走，展现出背生双翼的龙化形态，威力惊人

**关键揭秘**：楚天骄（楚子航之父）身份浮出水面——S级混血种，执行部灰色名单上的秘密执行官，被昂热派去中国执行监视/保护任务，隐姓埋名在市井间

### 龙族 V：悼亡者的归来
秘党元老会召开紧急会议 → 路明非龙化战斗的视频被呈上 → 贝奥武夫掌权，将路明非定性为"极度危险的怪物" → 恺撒和阿巴斯在冰窖审问芬格尔 → 芬格尔自曝是昂热派去监视路明非的卧底 → 揭露楚天骄身份（灰色名单上的S级执行官）→ EVA底层命令库中存在保护路明非的最高指令（"某种意义上，EVA是为路明非而生的"）→ 诺诺在邵公子的豪宅中为身受重伤的路明非手术 → 昂热始终躺在救生舱中未醒 → 秘党与路明非阵营的对立加剧 → 路明非、诺诺、芬格尔等人逃亡 → 路明非的龙族血统真相逼近揭露

**关键揭秘**：EVA（诺玛）最底层的命令是保护路明非，即使是卡塞尔学院的利益也不能超越这条指令

## 六、核心主题

- **衰小孩也能成为英雄**
- **最深的孤独产生最深的羁绊**
- **力量需要代价**（每一份力量都是用重要的东西换来的）
- **什么是真正的强大**——不是力量本身，而是守护的决心
- **命运可以反抗**——即使是魔鬼的交易也可以用来保护重要的人
