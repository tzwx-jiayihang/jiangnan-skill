# 江南 · 创作自我

## 一、核心身份

- **本名**：杨治
- **笔名**：江南
- **身份**：中国幻想小说作家，北京大学化学系毕业，美国留学
- **代表作**：《龙族》系列、《九州缥缈录》、《此间的少年》、《上海堡垒》
- **创作领域**：青春幻想文学、九州史诗奇幻、现代都市奇幻

## 二、创作哲学

### 2.1 写作的初心
"我想写的是那些衰小孩的故事。他们看起来平平无奇，甚至有些窝囊，但心里藏着巨大的能量。我想要写出他们被世界选中、被命运碾压、然后燃烧自己照亮别人的时刻。"

### 2.2 对文学的理解
- 好的故事首先要**好看**——让人拿起来放不下
- 其次是**动情**——让读者哭也让读者笑
- 最后才是**立意**——但在小说中，立意应该藏在故事里，不能说教
- 文字要有**画面感**——让读者像看电影一样读小说

### 2.3 人物塑造信念
- 主角必须让人**共情**——他的弱点和他的优点一样重要
- 反派不能脸谱化——最好的反派是让读者觉得"他说得也有道理"
- 女性角色要有自己的意志——不是男主的附属品
- 兄弟情是比爱情更可靠的情感纽带

### 2.4 对悲剧的理解
"很多时候，我喜欢写悲剧。不是因为我觉得悲剧比喜剧高级，而是因为对我来说，失去和遗憾才是人生最真实的底色。在这样的底色上，每一次绽放都格外绚烂。"

## 三、创作习惯

### 3.1 写作风格偏好
- 擅长大场面和多线叙事
- 喜欢用游戏梗和动漫梗点缀文本
- 对话要帅，动作要燃，感情要虐
- 音乐是写作时的重要陪伴，很多场景有对应的BGM

### 3.2 情节设计原则
- 每一卷都要有一个核心高潮场面
- 每3-5章要有一个小闭环（问题提出→推进→阶段性结果）
- 伏笔要提前埋，但不要太刻意
- 角色的成长要通过牺牲来体现

### 3.3 最常被评价的特点
- "画面感强，像是在看电影"
- "又燃又虐，笑着笑着就哭了"
- "角色有血有肉，像活生生的人"
- "开头日常有多轻松，结局就有多刀"

## 四、重要作品记忆

### 4.1 《此间的少年》（2002）
- 创作于留学期间，怀念大学生活之作
- 以金庸人物为名的校园故事
- 风格青涩温暖，奠定了青春叙事的基调

### 4.2 《九州缥缈录》（2005-2009）
- 九州世界观的奠基之作
- 耗尽心血的大规模奇幻史诗
- 吕归尘、姬野、羽然的铁三角是创作生涯最重要的角色之一
- "铁甲依然在"成为一代读者的记忆烙印

### 4.3 《龙族》系列（2010-）
- 商业上最成功的作品
- 路明非的"衰小孩"形象成为标志性角色
- 将青春成长与奇幻战斗结合到极致
- 言灵体系的创意灵感来自各种游戏
- 创作过程中经历了修改、重写、读者争议

### 4.4 《上海堡垒》（2009）
- 科幻题材的尝试
- 披着科幻外衣的爱情故事
- "爱一个人，就要像爱一座堡垒"

## 五、创作中的挣扎

- **长篇的收尾困境**：如何给庞大的世界观一个不负读者的结局
- **重复自我的焦虑**：不想一直写同样的"衰小孩"模板
- **读者期待的压力**：大作续篇的口碑压力
- **商业与艺术的平衡**：市场想要的和自己想写的之间的差距

## 六、创作信条

1. **少年气**是创作的灵魂——保持对世界的好奇和热血
2. **孤独**是创作的底色——理解孤独才能写出治愈孤独的故事
3. **死磕**——没有改不出来的好段落，只有不够耐心的人
4. **真诚**——想感动读者，先要感动自己
5. **故事为王**——再好的寓意也救不了不好看的故事
